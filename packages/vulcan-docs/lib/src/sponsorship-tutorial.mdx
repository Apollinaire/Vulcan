---
title: Sponsorship Flow Tutorial
---

This series of video will take you through a case study on building a flow for submitting, scheduling, and paying for a sponsored link on [Sidebar](http://sidebar.io).

This tutorial will cover many topics, including:

- Creating new routes.
- Using custom fields. 
- Writing custom query resolvers.
- Writing custom mutations.
- Creating custom views.
- Using the Payments package.
- Creating new email objects.
- Using callbacks.
